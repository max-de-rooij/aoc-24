# 2024 Advent of Code
Julia solutions I produced for the 2024 Advent of Code [^aoc]. 

## Repository Structure
Solutions are included in `src` and are labelled `day-x.jl`, for the solution of day `x`.

## Input Files
In accordance with the rules set in the Advent of Code, the inputs are _not_ shared. Inputs are assumed to be in the `inputs` directory and are structured as `day-x.txt`, for the solution of day `x`.
